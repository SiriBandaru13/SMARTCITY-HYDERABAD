## 📌 Overview

**Smart City Hyderabad** is a Java-based desktop application built using **Swing and AWT**, designed to help people who are new to Hyderabad easily explore the city.

The system provides information about **restaurants, hotels, educational institutions, and tourist places**, making it a one-stop solution for newcomers, students, and travelers.

---

## 🎯 Objective

- Help newcomers explore Hyderabad easily  
- Provide information about **restaurants, hotels, and tourist places**  
- Assist students in finding **universities and colleges**  
- Offer a simple and interactive desktop interface  

---

## 🚀 How It Works

1️⃣ User registers or logs into the system  
2️⃣ Selects required category:
   - Restaurants  
   - Hotels  
   - Tourist Places  
   - Colleges/Universities  
3️⃣ System fetches data from MySQL database  
4️⃣ Displays results in GUI
